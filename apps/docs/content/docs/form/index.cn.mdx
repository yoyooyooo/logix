`@logixjs/form` 是建立在同一条 Logix 主链上的领域包。

```text
Form.make(id, config, define)
  -> FormProgram
  -> Runtime / Program law
  -> FormHandle
  -> useModule + useSelector
```

## 公开路线

- 用 `Form.make(...)` author。
- 通过 core runtime 与 React host 挂载。
- 通过返回的 form handle 修改。
- 通过 `useSelector(...)`、`fieldValue(...)`、`fieldValues(...)`、`rawFormMeta()`、`Form.Error.field(...)`、`Form.Companion.*` descriptors 读取。

Form 不新增 Form-owned React hook family、公开 path-object API、公开 row-token API，也不制造第二套 runtime/control-plane truth source。
