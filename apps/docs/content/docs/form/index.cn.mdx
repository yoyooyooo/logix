Form 是返回 Logix `Program` 的领域包。它拥有可编辑 values、validation、source receipts、companion facts、submit 和 list identity，同时保持 React 读取走 core host route。

```text
Form.make(id, config, define)
  -> FormProgram
  -> Runtime / Program law
  -> FormHandle
  -> useModule + useSelector
```

## 路线

1. [Introduction](/cn/docs/form/introduction) 固定 owner map。
2. [Quick start](/cn/docs/form/quick-start) 构建一个小表单。
3. [Schema](/cn/docs/form/schema)、[Rules](/cn/docs/form/rules)、[Validation](/cn/docs/form/validation) 说明 final truth。
4. [Sources](/cn/docs/form/sources) 与 [Companion](/cn/docs/form/companion) 分离 remote facts 与 local soft facts。
5. [Field arrays](/cn/docs/form/field-arrays)、[Selectors](/cn/docs/form/selectors)、[Performance](/cn/docs/form/performance) 覆盖大型表单。

Form 不拥有 React hook family。React 集成仍然是 `useModule` 与 `useSelector`。
