Form is a domain package that returns a Logix `Program`. It owns editable values, validation, source receipts, companion facts, submit, and list identity while keeping React reads on the core host route.

```text
Form.make(id, config, define)
  -> FormProgram
  -> Runtime / Program law
  -> FormHandle
  -> useModule + useSelector
```

## Route

1. [Introduction](/docs/form/introduction) fixes the owner map.
2. [Quick start](/docs/form/quick-start) builds a small form.
3. [Schema](/docs/form/schema), [Rules](/docs/form/rules), and [Validation](/docs/form/validation) cover final truth.
4. [Sources](/docs/form/sources) and [Companion](/docs/form/companion) separate remote facts from local soft facts.
5. [Field arrays](/docs/form/field-arrays), [Selectors](/docs/form/selectors), and [Performance](/docs/form/performance) cover larger forms.

Form does not own a React hook family. React integration remains `useModule` and `useSelector`.
