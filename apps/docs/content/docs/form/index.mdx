`@logixjs/form` is a domain package built on the same Logix spine.

```text
Form.make(id, config, define)
  -> FormProgram
  -> Runtime / Program law
  -> FormHandle
  -> useModule + useSelector
```

## Public route

- Author with `Form.make(...)`.
- Mount with the core runtime and React host.
- Mutate with the returned form handle.
- Read with `useSelector(...)`, `fieldValue(...)`, `fieldValues(...)`, `rawFormMeta()`, `Form.Error.field(...)`, and `Form.Companion.*` descriptors.

Form does not add a Form-owned React hook family, public path-object API, public row-token API, or a second runtime/control-plane truth source.
