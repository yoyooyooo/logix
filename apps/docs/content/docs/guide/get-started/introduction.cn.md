Logix 是一个面向状态与业务逻辑的 Effect-native runtime。

它适合处理这些问题：

- 结构化状态
- 显式的异步编排
- 稳定的宿主集成
- 可诊断的执行过程

## 典型问题

- 快速 UI 变化带来的请求竞态
- 散落在组件里的业务逻辑
- 难以追踪的状态联动
- 不清楚异步工作为什么开始、停止或被替换

## 核心层次

Logix 可以先按 4 层理解：

- intent
- flow
- logic
- runtime

Guide 会按这个顺序逐步展开。
