Guide 按 Logix 应用的真实结构展开：定义 module，装配 program，运行 runtime，投影到 React，再按需要加入模式和控制面。

## 路线

| 区域 | 页面 |
| --- | --- |
| Get Started | [介绍](/cn/docs/guide/get-started/introduction)、[快速开始](/cn/docs/guide/get-started/quick-start)、搜索教程。 |
| Essentials | [Canonical spine](/cn/docs/guide/essentials/canonical-spine)、状态、Effect、React、生命周期、tick/flush。 |
| Patterns | 搜索详情、分页、乐观更新、i18n、表单向导。 |
| Advanced | 并发、Suspense、测试、性能、scope、排障。 |

每个区域都使用同一组对象角色：`Module` 定义，`Program` 装配，`Runtime` 执行，React 通过 `useModule` 与 `useSelector` 读取。
