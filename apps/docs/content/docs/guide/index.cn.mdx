## 这份 Guide 有意不教授的内容

- `useLocalModule`、`useModuleList` 和 `ModuleScope` 不是当前公开 React API 路线。
- Module 对象通过 `Program.make(...)` 装配；React 局部实例使用 `useModule(Program, options)`。
- Form 不定义包内 hook family；读取仍然使用 `useSelector(...)`。
- 验证报告来自 `Runtime.check(...)`、`Runtime.trial(...)`、`Runtime.compare(...)`，不是用户自定义 Form 或 React report 对象。
