## What this guide deliberately does not teach

- `useLocalModule`, `useModuleList`, and `ModuleScope` are not current public React API routes.
- Module objects are assembled through `Program.make(...)`; React local instances use `useModule(Program, options)`.
- Form does not define a package-local hook family. Reads remain `useSelector(...)`.
- Verification reports come from `Runtime.check(...)`, `Runtime.trial(...)`, and `Runtime.compare(...)`, not from user-authored Form or React report objects.
