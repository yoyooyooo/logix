| Pattern | Runtime owner |

| [Search + detail](/docs/guide/patterns/search-detail) | selected id, query state, request lifecycle |
| [Pagination](/docs/guide/patterns/pagination) | cursor/page state and loading windows |
| [Optimistic updates](/docs/guide/patterns/optimistic-update) | rollback state and effect result |
