Logix 是 Effect-native 的应用逻辑运行时。当前公开模型刻意保持很小：

```text
Module.logic(...)     definition-time logic 与声明
Program.make(...)     assembly-time business unit
Runtime.make(...)     execution-time runtime container
RuntimeProvider       React host projection
useModule + useSelector  React 实例获取与读取
```

本网站是用户手册。若要查看完整导出签名，使用生成的 [API Reference](/api-reference)。

## 当前公开面规则

当前文档不再教授已移除或内部路线，例如 `useLocalModule`、`useModuleList`、`ModuleScope`、公开 FieldKernel namespace、Form 自有 React hook family、公开 Form path 对象或第二套 runtime/control-plane facade。React 局部实例使用 `useModule(Program, options)` 表达；Form 读取仍然走 `useSelector(...)`。
