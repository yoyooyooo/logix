Logix is an Effect-native runtime for application logic. Its current public model is intentionally small:

```text
Module.logic(...)     definition-time logic and declarations
Program.make(...)     assembly-time business unit
Runtime.make(...)     execution-time runtime container
RuntimeProvider       React host projection
useModule + useSelector  React instance acquisition and reads
```

Use this site as the user-facing manual. For exhaustive exported signatures, use the generated [API Reference](/api-reference).

## Current public surface policy

The current docs do not teach removed or internal routes such as `useLocalModule`, `useModuleList`, `ModuleScope`, public FieldKernel namespaces, Form-owned React hook families, public Form path objects, or separate runtime/control-plane facades. Local React ownership is expressed with `useModule(Program, options)`. Form reads still use `useSelector(...)`.
