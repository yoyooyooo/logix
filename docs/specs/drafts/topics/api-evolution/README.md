# API Evolution · Topic Overview

本 Topic 专注于「下一代关键 API 形态」的探索，尤其是围绕 Bound API (`$`) 演进出的 Agent / AI / Flow 相关能力。

核心关注点：

- 以 `Agent.gen`、`$.whenAI` 等为代表的新 API 形态，如何在保持可解析性的前提下提升 DX；  
- 将 AI 调用从命令式 Effect 提升为声明式、一等公民的流式原语；  
- 为 v3/v4 以后的 Logix/平台 API 演进提供候选路径。
