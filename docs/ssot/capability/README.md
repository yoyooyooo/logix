# Logix Capability SSoT Root

## 目标

把 capability decomposition、API planning projection、proposal portfolio、principle promotion 与 proof-wave 的通用治理规则从 Form 子树上提到 Logix 全局层。

Form 继续是第一个高压 domain projection，但不持有全局方法论本体。

## 页面角色

- 本目录持有 Logix-wide capability planning harness
- 本目录定义所有 domain projection 必须继承的 coordinate、kernel、workflow 与 promotion lane
- 本目录不定义任何 domain exact surface
- 本目录不替代 `runtime/`、`form/`、`platform/` 的 owner authority
