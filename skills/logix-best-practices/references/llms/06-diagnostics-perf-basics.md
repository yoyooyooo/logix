# 诊断与性能证据（LLM 版）

## 1) 事件约束

- 事件模型 Slim，导出可序列化（`JsonValue`）。
- 关键链路保留稳定锚点：`instanceId/txnSeq/opSeq`。
- 发生降级时必须给出可解释原因。

## 2) 性能证据闭环

1. 固定采样条件（环境、样本、输入规模）。
2. 采集 before/after。
3. 生成 diff 与结论（至少 p50/p95）。
4. 附可复现命令与参数。

## 3) 结论门禁

以下结论无效：

- 只有体感，没有数据。
- 只有最终状态，没有过程证据。
- 关闭诊断开关后“看起来没成本”。
