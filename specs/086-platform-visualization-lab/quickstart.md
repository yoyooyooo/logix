# Quickstart: Platform Visualization Lab（086）

## 运行

在仓库根目录执行：

- `pnpm -C examples/logix-react dev`

然后打开本地地址（Vite 默认端口）：

- `http://localhost:5173/platform-viz`

## 体验路径

- `/platform-viz/manifest`：选择一个模块对象 → 查看 `ModuleManifest` 摘要与 Raw JSON（可一键复制）
- `/platform-viz/manifest-diff`：选择 before/after（或粘贴 JSON）→ 查看 verdict/summary/changes + Raw JSON
- `/trial-run-evidence`：运行浏览器侧 TrialRun 并查看 evidence.summary（本特性仅提供直达入口）
