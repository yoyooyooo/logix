## 规则

- 所有 reason code 必须在目录中登记。  
- 未登记 reason code 视为协议错误并 fail-fast。
- runtime gate 只允许：`GATE_TYPE_FAILED/GATE_LINT_FAILED/GATE_TEST_FAILED/GATE_CONTROL_SURFACE_ARTIFACT_FAILED/GATE_DIAGNOSTICS_FAILED`。
- governance gate 只允许：`GATE_PERF_HARD_FAILED/GATE_SSOT_DRIFT_FAILED/GATE_MIGRATION_FORWARD_ONLY_FAILED`。
- 旧命令迁移必须使用 `E_CLI_COMMAND_MERGED`，并在 `nextActions` 提供替代命令。
